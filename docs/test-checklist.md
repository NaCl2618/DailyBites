## 테스트 방법

1. 개발 서버 실행: `npm run dev`
2. http://localhost:3000 접속
3. 위 항목을 순서대로 테스트
4. 결과란에 `[x]` (통과) 또는 `[!]` (실패) 표시
5. 실패한 항목은 비고란에 에러 메시지나 현상 기록
6. 테스트 완료 후 이 파일을 저장하고 알려주세요
